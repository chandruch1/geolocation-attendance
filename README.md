---

## 🧠 Project Overview

The **Geolocation Based Attendance System** is a smart web application developed using **React.js** that automatically marks student attendance when their real-time GPS location matches the predefined college campus coordinates (latitude & longitude).

This system eliminates manual attendance processes and ensures secure, location-based validation.

---

## 🚀 Key Features

- 📍 Automatic attendance marking using GPS
- 🛰️ Real-time location detection via Browser Geolocation API
- 🎯 Campus boundary validation using latitude & longitude comparison
- ⚡ Fast and responsive UI built with React
- 🌐 Deployed on Vercel for live access

---

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| React.js | Frontend development |
| JavaScript | Core logic |
| HTML/CSS | UI structure and styling |
| Browser Geolocation API | Location tracking |
| Vercel | Deployment platform |

---

## ⚙️ How It Works

1. The application requests user permission to access location.
2. The browser fetches real-time latitude and longitude.
3. The system compares user coordinates with predefined campus boundary values.
4. If within the allowed radius → Attendance is automatically marked.
